### 데이터 & 정보 정의 ###

# 데이터(Data)
# : 사실이나 대상에 관한 수치, 문자 등의 표현
# : 가공되지 않은 원시적인 형태
# EX) 학생들의 성적, 고객의 주소, 판매된 상품의 수량, 핸드폰 사용량

# 정보(Information)
# : 데이터를 의미있게 가공하고 해석하여 결정에 도움을 줄 수 있는 형태로 변환한 결과물
# EX) 학생들의 성적 데이터를 분석해 전체 평균 성적을 구하는 과정에서 얻어진 전체 평균 성적

### 데이터베이스 정의 ###

# 데이터베이스(Database)
# : 조직에 필요한 여러 사용자들이 공유하여 사용할 수 있도록 통합, 저장된 연관된 정보들의 집합
# : 구조적인 방법으로 데이터를 저장, 관리, 검색, 수정할 수 있는 시스템
# : 통합하여 관리되는 데이터의 집합체 

### DBMS ###

# DBMS(Database Management System)
# : 데이터 베이스를 생성하고 관리(수정, 삭제, 검색)하는 소프트웨어

# DBMS의 역할
# : 데이터 저장 및 검색
# : 데이터 보안
# : 데이터 복구 및 백업

# DBMS의 종류
# 1. RDBMS(관계형 DBMS)
# : 데이터를 테이블 형태로 저장, 데이터 간의 관계를 테이블의 키를 통해 연결
# : Oracle, MySQL, MS SQL Server 등

# 2. NoSQL DBMS
# : 관계형 모델을 사용하지 않고, 대규모 분산 데이터를 저장하고 검색하는데 특화
# : MongDB, Redis 등
